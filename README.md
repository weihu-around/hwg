# hwg
模仿若依框架，从0开始做的一个后台管理系统

# 技术
vue3 + ts

# 功能
登录功能，用户管理列表，订单列表和订单详情，配置管理，用户个人中心，及用户修改密码，封装了axios请求，用于调试接口
